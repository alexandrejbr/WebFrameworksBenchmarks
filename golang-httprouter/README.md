## What was used
* [go] [go]
* [httprouter] [httprouter]

## how to run it

Follow these steps:

1. Install go
2. Define workspace path:

		$export GOPATH="golang-httprouter"

3. Install httprouter-hello:

		$go get httprouter-hello

3. Run:

		$./bin/httprouter-hello

4. Ready to take request @ http://127.0.0.1:8000/ (/text, /json)
