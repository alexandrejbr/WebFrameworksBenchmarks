## how to run it

Follow these steps:

1. Install go
2. Define workspace path:

		$export $GOPATH="golang-goji"

3. Install goji-hello:

		$go get goji-hello

3. Run:

		$./goji-hello

4. Ready to take request @ http://127.0.0.1:8080/ (/text, /json)
