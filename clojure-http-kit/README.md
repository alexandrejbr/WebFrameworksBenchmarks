## What was used

* [Leiningen][l]

## how to run it

Follow these steps:

1. Install Leiningen from http://leiningen.org/#install
2. Change directory into project's root directory
3. Run (using Leiningen):

		$lein trampoline run

4. Ready to take request @ http://127.0.0.1:8080/ (/text, /json)


[l]: http://leiningen.org/
